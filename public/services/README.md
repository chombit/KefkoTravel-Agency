# Services Section Images

Place the following images in this directory for the Services Section:

## Required Images (1200x800px recommended):

1. **flights.jpg** - Airplane/aviation/travel booking image
2. **hotels.jpg** - Luxury hotel/resort accommodation image  
3. **tours.jpg** - World travel/tourist destinations image
4. **visa.jpg** - Passport/visa documents/official travel image
5. **car-rental.jpg** - Car rental/transportation image
6. **insurance.jpg** - Travel protection/insurance concept image

## Image Guidelines:
- **Dimensions:** 1200x800px (3:2 ratio)
- **Format:** JPG or PNG
- **Quality:** High resolution, professional travel photography
- **Style:** Bright, inviting, travel-related imagery
- **Content:** Should represent each service category clearly

## Fallback:
If images are not available, the cards will show gradient backgrounds with the service icons.
