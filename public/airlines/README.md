# Airline Showcase Images

Place the following airline images in this directory for the Airline Showcase Section:

## Required Images (1200x600px recommended):

1. **ethiopian-airlines.jpg** - Ethiopian Airlines aircraft/livery
2. **emirates.jpg** - Emirates Airlines aircraft/livery  
3. **turkish-airlines.jpg** - Turkish Airlines aircraft/livery
4. **qr-airways.jpg** - Qatar Airways aircraft/livery
5. **lufthansa.jpg** - Lufthansa Airlines aircraft/livery

## Image Guidelines:
- **Dimensions:** 1200x600px (2:1 ratio) for main slider
- **Format:** JPG or PNG
- **Quality:** High resolution, professional aviation photography
- **Style:** Clear aircraft images, preferably in flight or at airport
- **Content:** Should show the airline's distinctive livery/branding
- **Lighting:** Bright, clear, professional aviation shots

## Fallback:
If images are not available, the slider will show gradient backgrounds with airline information.
